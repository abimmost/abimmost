### [Toolify](https://github.com/iwstech3/Toolify) - Main Contributor
An AI-driven tool recognition and manual generation platform designed to identify tools from images and generate comprehensive user manuals, safety guides, and research insights. Built with a powerful FastAPI backend and modern Next.js frontend.

- **Tech Stack:** TypeScript (59.8%), Python (35.8%), CSS (4.4%), FastAPI, Next.js 15, Gemini Vision, Tavily API, Supabase, Clerk Auth
- **Key Features:**
  - **AI Tool Recognition** – Gemini Vision analyzes uploaded images to accurately identify tools or machinery
  - **Intelligent Manual Generation** – Automatically generates detailed user manuals, quick-start guides, and safety instructions for recognized tools
  - **Audio Support** – Converts generated manuals to MP3 using gTTS for accessibility and on-the-go learning
  - **Real-Time Research** – Tavily AI fetches latest usage tips, maintenance guides, and YouTube tutorial links
  - **Cloud Storage** – Securely stores captured images and user scan history in Supabase
  - **Safety Guides** – Structured safety precautions and PPE recommendations
- **Backend Architecture:** FastAPI with LangChain orchestration, Supabase authentication & storage
- **Frontend:** Next.js 15 with TypeScript, Tailwind CSS, Clerk authentication
- **Live Demo:** https://toolify-zeta.vercel.app
